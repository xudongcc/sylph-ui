## Installation

```bash
pnpm dlx shadcn@latest add @sylph-ui/page
```

## Usage

```tsx
import { Page } from "@/components/sylph-ui/page";
```

```tsx
<Page title="Products" description="Manage your products and inventory.">
  {/* Page content */}
</Page>
```
