## Installation

```bash
pnpm dlx shadcn@latest add @sylph-ui/code-block
```

## Usage

```tsx
import {
  CodeBlock,
  CodeBlockBody,
  CodeBlockContent,
  CodeBlockCopyButton,
  CodeBlockHeader,
  CodeBlockItem,
} from "@/components/sylph-ui/code-block";
```

```tsx
const data = [
  {
    language: "tsx",
    filename: "example.tsx",
    code: "console.log('Hello, World!');",
  },
];

<CodeBlock data={data} defaultValue="tsx">
  <CodeBlockHeader>
    <span className="text-muted-foreground px-3 text-xs">example.tsx</span>
    <CodeBlockCopyButton className="ml-auto" />
  </CodeBlockHeader>
  <CodeBlockBody>
    {(item) => (
      <CodeBlockItem key={item.language} value={item.language}>
        <CodeBlockContent language="tsx">{item.code}</CodeBlockContent>
      </CodeBlockItem>
    )}
  </CodeBlockBody>
</CodeBlock>;
```

## Features

- **Syntax Highlighting** - Powered by Shiki with support for 100+ languages
- **Dark Mode** - Automatic theme switching with light and dark themes
- **Line Numbers** - Optional line numbers display
- **Copy Button** - One-click code copying to clipboard
- **File Tabs** - Support for multiple files with tabbed navigation
- **File Icons** - Automatic file type icons based on filename
