## Installation

```bash
pnpm dlx shadcn@latest add @sylph-ui/contextual-save-bar
```

## Usage

```tsx
import {
  ContextualSaveBar,
  ContextualSaveBarMessage,
  ContextualSaveBarActions,
  ContextualSaveBarDiscard,
  ContextualSaveBarSave,
} from "@/components/sylph-ui/contextual-save-bar";
```

```tsx
<ContextualSaveBar>
  <ContextualSaveBarMessage>Unsaved changes</ContextualSaveBarMessage>
  <ContextualSaveBarActions>
    <ContextualSaveBarDiscard>Discard</ContextualSaveBarDiscard>
    <ContextualSaveBarSave>Save</ContextualSaveBarSave>
  </ContextualSaveBarActions>
</ContextualSaveBar>
```

## Best Practices

- Only show when unsaved changes exist
- Use short, helpful messages following the "adjective + noun" pattern (e.g., "Unsaved changes")
- Save all changes on the page simultaneously; avoid multiple independent save bars on a single page
- Use strong action verbs ("Save", "Discard") instead of nouns

## API Reference

### ContextualSaveBar

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| fullWidth | `boolean` | `false` | Removes the default max-width constraint |
| children | `ReactNode` | - | Content of the save bar |

### ContextualSaveBarMessage

Container component for displaying the message.

### ContextualSaveBarActions

Container component for action buttons.

### ContextualSaveBarDiscard

Discard button based on the Button component, using the `outline` variant by default.

### ContextualSaveBarSave

Save button based on the Button component, using the `default` variant by default.
