## Installation

```bash
pnpm dlx shadcn@latest add @sylph-ui/dialog
```

## Usage

```tsx
import {
  Dialog,
  DialogClose,
  DialogContent,
  DialogDescription,
  DialogFooter,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from "@/components/sylph-ui/dialog";
```

```tsx
<Dialog>
  <DialogTrigger>Open</DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>Dialog Title</DialogTitle>
      <DialogDescription>Dialog description goes here.</DialogDescription>
    </DialogHeader>
    <div>Your content here</div>
  </DialogContent>
</Dialog>
```

## Features

- **Responsive Design**: Automatically adapts to screen size
  - Desktop: Renders as a centered modal dialog
  - Mobile: Renders as a bottom drawer for better touch interaction
- **Seamless API**: Uses the same component API regardless of device type
- **Built on Radix UI**: Leverages Dialog and Drawer primitives for accessibility

## Best Practices

- Use clear, descriptive titles that explain the purpose of the dialog
- Keep dialog content focused and concise
- Provide a clear way to dismiss the dialog (close button or click outside)
- Use `DialogDescription` to provide additional context for screen readers
