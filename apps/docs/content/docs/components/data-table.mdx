## Installation

```bash
pnpm dlx shadcn@latest add @sylph-ui/data-table
```

## Usage

```tsx
import { DataTable, type DataTableColumnProps } from "@repo/data-table";
```

```tsx
interface User {
  id: string;
  name: string;
  email: string;
}

const columns: DataTableColumnProps<User>[] = [
  {
    id: "name",
    header: "Name",
    accessorKey: "name",
  },
  {
    id: "email",
    header: "Email",
    accessorKey: "email",
  },
];

const data: User[] = [
  { id: "1", name: "Alice", email: "alice@example.com" },
  { id: "2", name: "Bob", email: "bob@example.com" },
];

<DataTable columns={columns} data={data} />;
```

## Features

- **Row Selection** - Single and multi-row selection with bulk actions support
- **Row Actions** - Dropdown menu with customizable actions per row
- **Column Pinning** - Pin columns to left or right for better visibility
- **Pagination** - Built-in pagination controls
- **Row Click** - Handle row click events for navigation or detail views
- **Flexible Columns** - Powered by TanStack Table with full column customization
